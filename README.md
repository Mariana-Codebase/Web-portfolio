# Mariana Dev - Portfolio

Neo‑Brutalist minimalist portfolio with a 3D background, terminal mode, bilingual content, and GitHub‑synced projects.

## Quick Start

```bash
npm install
npm run dev
```

## Tech

- React 18 + TypeScript
- Vite
- Three.js
- Tailwind CSS
- React Three Fiber
